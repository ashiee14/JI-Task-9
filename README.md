# JI-Task-9

StudentManagementApp

Demonstrates:
1. Store Student objects in ArrayList.
2. Use HashMap for fast lookup by student ID.
3. Implement sorting using Comparator (by marks and name).
4. Remove duplicates using Set.
5. Iterate using enhanced for loops.
6. Demonstrate collection hierarchy.
7. Optimize memory usage by avoiding duplicates.
8. Print formatted reports using printf.
